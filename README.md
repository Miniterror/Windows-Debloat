Script made for personal usage to run after installation of Windows 11 Pro 25H2.
Also have a version with toggles for certain questions like apps to install, powerscheme settings, dark mode and a few other things.

Silent version can be run from Powershell in ADMIN mode.
irm https://miniterror.nl/FTS | iex

Toggle version can be started from Powershell in ADMIn mode.
irm https://miniterror.nl/FTStoggles | iex
